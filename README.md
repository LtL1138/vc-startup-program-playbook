# VC + Startup Program Playbook (DevTools / Infrastructure)

Lightweight playbooks and templates for building and scaling VC partnerships and startup programs for a developer-first infrastructure product.

## Who this is for
Partnerships, ecosystem, and growth leaders working with:
- Venture firms (GP/MD level)
- Accelerators and founder communities
- Seed–Series C startups making infrastructure decisions

## What’s inside
- Playbooks: account planning, program ops, escalation triage, product feedback loops
- Templates: outreach, portfolio intros, demo day follow-up
- Metrics: adoption, retention, and partner success measures

## Principles
Founder-first, build for scale, async-first documentation, measurable impact.
